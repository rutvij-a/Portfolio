Smart Health Monitoring System

Project Overview

The Smart Health Monitoring System Using IoT is designed to track and monitor essential health parameters remotely using IoT technology. With the increasing demand for efficient healthcare solutions, this system enables real-time health tracking for elderly individuals or patients requiring continuous monitoring. The main objective is to collect health data such as heart rate, blood oxygen level (SpO2), temperature, and ECG using various biomedical sensors. This data is uploaded to an IoT cloud platform (ThingSpeak), making it accessible to doctors and caregivers remotely. The system ensures timely diagnosis and alerts in case of abnormal readings, improving healthcare efficiency.

Features & Functionality

  • Multi-Sensor Integration: The system utilizes ECG (AD8232), Heart Rate & SpO2 (MAX30102), and Temperature (DS18B20) sensors to continuously track health metrics.

  • IoT-Based Data Upload: Health readings are uploaded to ThingSpeak Cloud, allowing remote access for doctors and caregivers.

  • LCD Display for Local Readings: Users can view real-time health statistics on an LCD display.

  • Automated Emergency Alerts: If health readings exceed critical levels (e.g., HR < 50 or > 100, SpO2 < 95, Temperature > 40°C), the system triggers a buzzer alert and displays an Emergency Warning.

  • Two-Mode System:

    • Mode 1 → Monitors Heart Rate, Blood Oxygen, and Temperature

    • Mode 2 → Captures ECG data for detailed heart activity

  • ESP8266 Wi-Fi Module Integration: Ensures seamless data transfer from sensors to the cloud.

  • User-Friendly & Wearable Design: The system is compact and suitable for daily use, enhancing healthcare accessibility.

Project Demonstration

  • The ECG sensor (AD8232), Heart Rate & SpO2 sensor (MAX30102), and Temperature sensor (DS18B20) collect health data.

  • The LCD display shows real-time readings. If values are abnormal, ‘Emergency’ is displayed, and a buzzer alerts the user.

  • The ESP8266 Wi-Fi module uploads data to the ThingSpeak IoT cloud, allowing remote tracking.

  • Doctors or caregivers can access health records remotely, ensuring timely medical intervention.

  • The system efficiently diagnoses critical conditions, reducing hospital visits and improving real-time health monitoring.
