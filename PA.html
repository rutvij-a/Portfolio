Python Assistant(ZEN)

Project Overview

The Python Assistant project, ZEN 1.0, is designed to act as a personal assistant with voice-based interaction capabilities. The goal is to create an intelligent software agent that responds to verbal commands, performs various automated tasks, and assists users efficiently. Unlike standard voice assistants, ZEN 1.0 goes beyond basic functionality by integrating features such as email handling, web browsing, media control, note-taking, and even system operations. The project aims to simplify daily activities through natural language processing and automation while maintaining user convenience and security.

Features & Functionality

ZEN 1.0 offers a range of functionalities, including:

  • Voice Command Recognition: Users can interact using verbal instructions.

  • Time & Date Retrieval :Provides current date, time, and weekday.

  • System Operations: Includes locking the screen, logging out, restarting, shutting down, and emptying the recycle bin.

  • Application & Web Search: Can search Wikipedia, Google, and open specific websites or applications like MS Word, CodeBlocks, and Eclipse.

  • Media Control: Plays songs, videos, and opens images.

  • Weather & News Updates: Retrieves real-time weather information and top headlines.

  • Camera & Recording Functions: Captures photos, records videos, and audio clips.

  • Mathematical Calculation: Solves equations and retrieves answers from Wolfram Alpha.

  • Email & Note-Taking: Sends emails and allows users to create and view notes.

Project Demonstration

The system is developed using Python with various modules for speech recognition, task automation, and media interaction. On activation, ZEN 1.0 welcomes the user, retrieves system time and date, and waits for voice commands. When a command is issued, it processes the request using the appropriate module and executes the task accordingly. The assistant ensures smooth interaction by responding audibly with results, making the experience more intuitive and engaging.
