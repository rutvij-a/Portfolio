Android Based Electric Switch

Project Overview

The Android-Based Electric Switch project aims to provide a remote-controlled electrical switch system using an Android app and Bluetooth communication. The objective is to make home automation more convenient and energy-efficient, allowing users to monitor and control appliances wirelessly. The system is built using an Arduino microcontroller, a Bluetooth module (HC-05), and a relay, ensuring seamless interaction between hardware and software.

Features & Functionality

  • Wireless Control: Users can switch electrical appliances ON/OFF remotely using an Android application.

  • Bluetooth Communication: The system uses Bluetooth technology for seamless communication between the Android device and the Arduino microcontroller.

  • Arduino Integration: The Arduino Uno processes commands and controls the electrical switch.

  • Relay Control: A relay module is used to switch AC appliances ON or OFF based on the app's input.

  • Energy Efficiency: Helps in reducing power consumption by remotely managing electrical devices.

Project Demonstration

  • The HC-05 Bluetooth module communicates with the Arduino Uno, allowing users to control devices via the ABES app.

  • When the user presses the power button in the app, the system checks the relay’s status:

    • If OFF → It turns ON the appliance.

    • If ON → It turns OFF the appliance.

  • The relay is responsible for physically switching the device ON or OFF.

  • The circuit is powered by a battery, ensuring consistent operation.
